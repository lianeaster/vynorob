# Винороб - Система створення технологічних схем

Веб-додаток для створення технологічних схем виноробства.

## Можливості

- Створення принципової технологічної схеми вина
- Крок за кроком вибір параметрів вина:
  - Колір (біле, червоне, рожеве)
  - Стиль (сухе, напівсухе, напівсолодке, солодке)
  - Стиль за CO2 (тихе, ігристе)
  - Кондиції сировини (цукор, кислотність, pH)
  - Кондиції вина (цукор, кислотність, pH, спиртуозність)
  - Вибір: Опис вина АБО Схема виробництва
    - Опис вина (характеристики: повнотіле, легке, танінне, фруктове, квіткове)
    - Схема виробництва (біла, червона, змішана)
- Збереження схем у базу даних
- Два варіанти збереження:
  - **Save** - зберегти схему
  - **Save and Create** - зберегти та перейти до створення технологічної схеми

## Технології

- **Backend:** Python + Flask
- **Frontend:** HTML, CSS, JavaScript
- **База даних:** JSON файли (non-relational)

## Встановлення

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Запустіть додаток:
```bash
python app.py
```

3. Відкрийте браузер за адресою: http://localhost:5000

## Використання

1. Натисніть "Почати" на головній сторінці
2. Виберіть колір вина
3. Виберіть стиль вина
4. Виберіть стиль за CO2
5. Введіть кондиції сировини
6. Введіть кондиції готового вина
7. Оберіть: "Опис вина" або "Схема"
   - Якщо "Опис вина" → виберіть характеристики
   - Якщо "Схема" → виберіть тип схеми (біла/червона/змішана)
8. Перегляньте результат
9. Оберіть:
   - **Save** - просто зберегти
   - **Save and Create** - зберегти і перейти до створення технологічної схеми

## API Endpoints

- `GET /` - Головна сторінка
- `GET /color` - Вибір кольору
- `GET /style` - Вибір стилю
- `GET /style-co2` - Вибір стилю за CO2
- `GET /raw-material` - Введення кондицій сировини
- `GET /wine-conditions` - Введення кондицій вина
- `GET /wine-description` - Опис вина
- `GET /create-scheme` - Створення схеми
- `POST /api/save-choice` - Збереження вибору
- `GET /api/get-session` - Отримання даних сесії
- `POST /api/save-wine` - Збереження схеми вина
- `GET /api/wines` - Отримання всіх схем
