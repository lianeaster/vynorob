# Винороб - Система створення технологічних схем

Веб-додаток для створення технологічних схем виноробства.

## Можливості

- Створення принципової технологічної схеми вина.
- Крок за кроком вибір параметрів вина:
  - **Крок 1:** Сировина, екстракція, пресування та схема бродіння (біла, червона, рожева, orange, blanc de noirs).
  - **Крок 2:** Солодкість, цільова спиртуозність, початковий цукор сусла (S₀), залишковий цукор (RS).
  - **Крок 3:** Стиль за CO₂ (тихе, перлант, фріззанте, ігристе), тиск, метод отримання CO₂, тара, додатковий цукор на вторинне бродіння.
  - **Крок 4:** Титрована кислотність (TA): поточна, цільова, режим корекції та відповідність нормам ДСТУ.
  - **Крок 5:** Бажаний смаковий стиль (баланс, структура, ароматичний профіль, дуб, кисень, MLF, кремовість).
  - **Крок 6:** Технічні параметри та стабільність (pH, YAN, SO₂, температура бродіння, осад, стабілізації, мікробіологічні ризики).
  - **Крок 7:** Підсумок та вибір типу схеми (біла/червона) перед побудовою технологічної схеми.
- Збереження схем у базу даних.

## Технології

- **Backend:** Python + Flask
- **Frontend:** HTML, CSS, JavaScript
- **База даних:** JSON файли (non-relational)

## Встановлення

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Запустіть додаток:
```bash
python app.py
```

3. Відкрийте браузер за адресою: http://localhost:5000

## Використання

1. Натисніть «Почати» на головній сторінці.
2. Виберіть сировину, екстракцію, пресування та схему бродіння (Крок 1).
3. Задайте солодкість, спиртуозність та початковий цукор сусла (Крок 2).
4. Оберіть стиль за CO₂ і параметри тиску/методу/тари (Крок 3).
5. Введіть кислотність (TA) та режим корекції (Крок 4).
6. Опишіть бажаний смаковий стиль (Крок 5).
7. Налаштуйте технічні параметри та стабільність (Крок 6).
8. Оберіть тип схеми (біла/червона) та перегляньте підсумок (Крок 7).
9. Створіть технологічну схему.

## API Endpoints

- `GET /` - Головна сторінка
- `GET /color` - Вибір кольору
- `GET /style` - Вибір стилю
- `GET /style-co2` - Вибір стилю за CO₂
- `GET /raw-material` - Введення кислотності сировини (TA)
- `GET /style-sensory` - Бажаний смаковий стиль
- `GET /style-tech` - Технічні параметри та стабільність
- `GET /scheme-type` - Тип схеми (біла/червона)
- `GET /summary` - Підсумок перед створенням схеми
- `POST /api/save-choice` - Збереження вибору на кроках
- `GET /api/get-session` - Отримання даних сесії
- `POST /api/save-wine` - Збереження повної схеми вина
- `GET /api/wines` - Отримання всіх схем
